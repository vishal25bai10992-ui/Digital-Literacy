# Digital-Literacy
Project Overview

This Digital Literacy Project is designed to build awareness and practical skills required to navigate the digital world effectively. As a Student Digital Ambassador, the project focuses on helping peers understand safe, responsible, and professional use of digital platforms.

The project consists of five modules covering key aspects such as digital awareness, online presence, collaboration tools, communication etiquette, and cyber safety. Each task is documented in a structured GitHub repository along with reflections and practical outputs

Module Summary
🔹 Module 1: Digital Literacy Awareness

Created a one-page infographic explaining digital literacy, its importance, and safe internet practices using design tools like Canva.

🔹 Module 2: Student Digital Portfolio

Developed professional profiles on platforms such as GitHub and LinkedIn to build an online presence and showcase academic growth.

🔹 Module 3: Coding & Collaboration Platforms

Explored coding platforms (e.g., HackerRank) and created a Google Form quiz to understand collaboration and data collection tools.

🔹 Module 4: Email Etiquette & Communication

Drafted professional emails and prepared a social media Do’s & Don’ts checklist to promote responsible digital communication.

🔹 Module 5: Cybercrime Awareness

Researched a cybercrime case study and created a prevention checklist to promote safe online behavior among students.
